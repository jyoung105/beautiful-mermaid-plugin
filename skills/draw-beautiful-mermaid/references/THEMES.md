# Built-in Themes

| Theme Name (--theme arg) | Type | Description |
|---|---|---|
| `zinc-light` | Light | Default light theme |
| `zinc-dark` | Dark | Default dark theme |
| `tokyo-night` | Dark | Deep blue background, neon accents |
| `tokyo-night-storm` | Dark | Slightly lighter background than regular Tokyo Night |
| `tokyo-night-light` | Light | Light version of Tokyo Night |
| `catppuccin-mocha` | Dark | Soft, high contrast dark theme |
| `catppuccin-latte` | Light | Soft, warm light theme |
| `nord` | Dark | Arctic blue-grey palette |
| `nord-light` | Light | Light arctic palette |
| `dracula` | Dark | High contrast, purple accents, very popular |
| `github-light` | Light | Classic GitHub light theme |
| `github-dark` | Dark | Classic GitHub dark theme |
| `solarized-light` | Light | Low contrast, yellowish light theme |
| `solarized-dark` | Dark | Low contrast, teal-based dark theme |
| `one-dark` | Dark | Atom's iconic dark theme |

## Custom Themes

You can also pass custom JSON objects if using the library programmatically.
